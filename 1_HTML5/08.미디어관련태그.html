미디어 관련 태그

HTML5 부터는 플러그인 기능을 사용하지 않고
미디어를 직접 HTML문서에서 실행시킬 수 있는 태그 제공.

audio 태그

            src : 경로
            controls : 재생도구 출력 지정
            autoplay : 자동 재생 여부 지정(모바일 적용 X)
            loop : 반복 여부 지정
            preload : 선로드 여부 지정

video 태그

            src : 경로
            controls : 재생도구 출력 지정
            autoplay : 자동 재생 여부 지정(모바일 적용 X)
            loop : 반복 여부 지정
            preload : 선로드 여부 지정
            poster : 재생 전 출력 이미지(썸네일)
            width / height
