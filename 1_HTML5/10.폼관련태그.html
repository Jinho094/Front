폼 관련 태그

input 관련 태그 : html에서 사용자가 입력할 수 있는 양식을 제공하는 태그

form 태그 : 내부에 작성 된 input 태그들을 통해 사용자로 부터 데이터를 입력 받아
서버로 넘기는 역할(전송)을 하는 태그. 

            form 태그 속성

            *action 속성 : form태그 내부에 입력 된 데이터를 전달 받아 처리할
                           페이지 또는 서버의 URL을 지정하는 속성.
            
            *method 속성 : 데이터 전송 방식을 지정하는 속성 
                - get : URL(주소창)에 데이터를 담아 전달하는 방식(데이터를 볼 수 있음)
                        데이터 크기에 제한이 있음.(256 ~ 4096byte)
                        
                - post : http request에 데이터를 넣어 전송하는 방식.(데이터를 볼 수 없음)
                         데이터 크기 제한 없음.

            *name 속성 : form 태그의 고유 이름 지정. form 태그 구분을 위해 사용.

            *target : action 속성에 작성 된 페이지를 어떻게 열 것인지 지정.

text 관련 input 태그

type = "text"

- 한 줄 짜리 텍스트를 입력할 수 있는 텍스트 입력 상자.

type = "password"

- 비밀번호를 입력할 수 있는 입력 상자.

type = "search" / "url" / "email" / "tel"

- 기본적인 형태를 입력상자와 같지만, 각각의 정보를 입력받을 수 있게 분화 된 기능을 제공함.





숫자와 관련된 input 태그

type = "number"

- 입력창에 숫자만 입력하게 하는 타입
- 브라우저에 따라 스핀박스가 표시되기도 함.
-- 브라우저 : HTML과 같은 Text 데이터를 해석해서 원래의 형태로 보여주는 프로그램
(스핀박스 : 입력창 오른쪽에 위, 아래 화살표 버튼)

type = "range"

- 슬라이드바를 통해 숫자 지정하는 타입.

중복 선택이 되지 않는 버튼

- 여러 태그가 같은 name 속성을 사용해도 중복 체크가 가능함. -> 서버에 데이터 전달 시, 체크 된 모든 데이터를 배열 형태로 전달함.
