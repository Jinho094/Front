CSS란?

Cascading Style Sheets 약자로 문서에 종속 된 스타일을 지정하는 시트
(마크업 언어가 실제 표시되는 방법을 기술하는 언어)

선택자란?

특정한 HTML 태그를 선택할 때 사용하는 기능으로
태그를 선택하여 원하는 스타일과 기능을 적용할 수 있다.

CSS 기본 선택자

모든 요소(태그) 선택자(*)

HTML문서 안에 모든 태그를 선택할 때 사용

태그 선택자(태그명)

HTML 문서 내에 같은 태그를 모두 선택할 때 사용

            p{ color : red } 와 같은 형태로 사용 됨

아이디 선택자(#)

HTML 문서 내에 해당하는 아이디 속성의 값을 가진 태그만 선택할 때 사용

*HTML 문서 내에서 태그별 id 속성 값은 유일해야 함(중복 X) 

            #아이디 속성 값 { color : blue }

  5. 아이디 선택자 테스트5

클래스 선택자( . )

HTML문서 내에서 여러 태그를 동일한 클래스명으로 지정할 경우 동일한 스타일을 부여할 수 있음. 

            .클래스명 { color : red }와 같은 형태로 사용
