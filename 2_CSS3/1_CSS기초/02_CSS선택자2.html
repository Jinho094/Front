            선택자[속성 = 속성값] { 설정 내용; }

자손, 후손 선택자

자손 선택자 : 선택 된 태그 바로 하위 요소
후손 선택자 : 선택 된 태그 하위 모든 요소 

            자손 선택자 : 선택자>선택자 { 설정 내용; }
                                    ㄴ 자식 선택자
                            (선택자1 바로 하위에 있는 선택자2)

            후손 선택자 : 선택자1 선택자2 { 설정 내용; }
                            (선택자1 하위에 있는 모든 선택자2)

동위 선택자

동위 관계에서 뒤에 위치한 태그를 선택할 때 사용

            바로 뒤에 요소를 선택할 때 : 선택자1 + 선택자2 { 설정내용; }
            뒤에 있는 모든 요소 선택할때는 선택자1 ~ 선택자2 { 설정 내용; }

반응 선택자

사용자의 움직임에 따라 달라지는 선택자

            선택자 : active { 설정 내용 }
            -> 사용자가 해당 선택자를 클릭했을 때 설정 내용으로 스타일이 변경

            선택자 : hover {  설정 내용 }
            -> 사용자가 해당 선택자에 마우스를 올려 놓았을 때 설정 내용으로 스타일이 변함

입력양식에 따른 선택자(상태 선택자)

            checkbox 또는 radio input 태그가 선택이 되었을경우
            스타일을 변경할 때 사용함.
            
            input태그 선택자:checkd { 설정 내용; }

초점이 맞추어진 input태그 선택자

            input태그 선택자: focous { 설정 내용; }

활성, 비활성 선택자

            사용 가능한 태그 선택 : input 태그 선택자:enabled { 설정 내용;}
            사용 불가능한 태그 선택 : input 태그 선택자:disabled { 설정 내용;}
