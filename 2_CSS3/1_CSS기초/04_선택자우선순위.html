선택자 우선 순위

            기본적으로 CSS 속성은 작성 된 순서대로(위 -> 아래)적용 되지만
            같은 태그에 여러개의 CSS속성이 설정 된 경우에는 
            선택자 우선 순위에 따라 스타일이 적용된다.

            1 순위 : !important;
            2 순위 : inline style 속성
            3 순위 : id 선택자
            4 순위 : 클래스 선택자
            5 순위 : 태그 선택자
