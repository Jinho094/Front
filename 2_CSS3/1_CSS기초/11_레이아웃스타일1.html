화면 배치 방법 변경

block, inline요소의 속성을 변경하여 배치 방식을 변경할 수 있다

block 요소를 inline 요소로 변경

            display : inline;
            지정 된 요소의 형석이 inline으로 변경되어
            화면을 수평분할 하고, width, height 속성이 무시된다

inline요소를 block 요소로 변경

            display : inline-block;
            영역 분할 방법은 inline 요소의 방법인 수평 분할로 지정 되지만
            content부분은 block요소로 지정되어
            width,height속성 지정이 가능함.
